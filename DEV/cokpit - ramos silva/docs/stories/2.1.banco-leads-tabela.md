## Story
**As a** André (prospector),
**I want** ver todos os leads da PGFN em uma tabela filtrável,
**so that** eu possa identificar rapidamente quais empresas valem trabalhar.

## Acceptance Criteria
1. Página `/banco-leads` exibe tabela com colunas: Nome, CNPJ, Valor Dívida, UF, Classificação, Status, Data Entrada
2. Classificação exibida como badge colorido: A (vermelho), B (amarelo), C (cinza)
3. Paginação de 100 registros por página com navegação anterior/próxima
4. Filtros funcionam combinados simultaneamente: UF + Classificação + Status + busca por texto
5. Busca por nome aceita texto parcial (case-insensitive); busca por CNPJ aceita com e sem máscara
6. Ordenação por Valor Dívida (desc por padrão, toggle asc/desc ao clicar no cabeçalho)
7. Route Handler `GET /api/leads` retorna dados paginados com filtros via query params
8. Loading state exibido enquanto dados carregam; erro exibido se requisição falhar

**Supabase query com filtros:**
```ts
let query = supabase.from('leads').select('*', { count: 'exact' })
if (uf) query = query.eq('uf', uf)
if (classificacao) query = query.eq('classificacao', classificacao)
if (status) query = query.eq('status', status)
if (q) query = query.or(`nome_empresa.ilike.%${q}%,cnpj.ilike.%${cnpjLimpo}%`)
query = query.order('valor_divida', { ascending: order === 'asc' })
query = query.range(offset, offset + limit - 1)
```

**Formatação de valores:**
```ts
// formatters.ts
export const formatCNPJ = (cnpj: string) =>
  cnpj.replace(/(\d{2})(\d{3})(\d{3})(\d{4})(\d{2})/, '$1.$2.$3/$4-$5')

export const formatMoeda = (valor: number) =>
```

### Testing
