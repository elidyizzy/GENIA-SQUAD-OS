## Dev Notes
**Stack:** Next.js 14 App Router + TypeScript + TailwindCSS + shadcn/ui + Supabase JS SDK

**Supabase clients:**
- Browser client: para componentes React client-side
- Server client: para Route Handlers (`app/api/**/route.ts`) — usa `SUPABASE_SERVICE_KEY`

**Schema completo:** ver `docs/SPEC-TECNICO.md` seção 4 (Schema do Banco de Dados)

**Variáveis de ambiente necessárias:**
```
SUPABASE_URL
SUPABASE_SERVICE_KEY         # só servidor
NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
```

**Path alias:** todas as imports usam `@/` — nunca caminhos relativos `../`
